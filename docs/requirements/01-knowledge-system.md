# 1. 知识体系展示

## 功能定位
构建层次化的知识体系导航系统，帮助学生清晰了解所学内容的整体结构和学习进度。

## 1.1 年级选择器

**功能特性**
- 支持按年级维度展示不同学期的知识体系
- 年级层级：七年级（上册/下册）、八年级（上册/下册）、九年级（上册/下册）
- 切换年级时平滑过渡，保留用户浏览位置
- 显示当前年级的知识点统计信息（模块数、专题数、知识点总数）

**交互设计**
- 顶部横向滚动标签页
- 当前选中年级高亮显示
- 显示当前年级的完成进度

## 1.2 层次化知识导航

采用四层结构展示知识体系：

### 第一层：课程概览
- 显示课程名称（如"初中数学七年级上册压轴题体系"）
- 展示整体学习进度条（已完成知识点/总知识点）
- 提供快速搜索功能，支持按模块/专题/知识点名称搜索

### 第二层：模块列表
卡片式展示所有模块，每个模块卡片包含：
- 模块名称（如"第一模块：数与式的深度运算"）
- 模块标签（如"代数思维篇"）
- 模块概述（简短描述，不超过50字）
- 包含的专题数量
- 学习进度指示器

**交互特性**
- 点击卡片展开/收起该模块下的专题列表
- 支持模块折叠/展开全部

### 第三层：专题列表
列表式展示模块下的所有专题，每个专题项包含：
- 专题名称（如"绝对值化简"）
- 专题别名（如"专题1"）
- 包含的知识点数量
- 学习状态标记（未开始/进行中/已完成）

**交互特性**
- 点击展开查看该专题的知识点详情
- 支持按学习状态筛选专题

### 第四层：知识点详情
展示具体的知识点内容，每个知识点包含：
- 知识点名称（如"零点分段法"）
- 知识点ID（如"KP01_1"）
- 详细说明文字
- 相关题目数量（可点击跳转到题目列表）
- 学习记录（首次学习时间、最后复习时间、掌握程度）

**交互特性**
- 支持收藏知识点功能
- 提供"开始学习"按钮，跳转到学习界面

## 1.3 交互设计要求

### 展开/收起动画
- 使用平滑的手风琴效果（transition: 300ms ease-out）

### 进度可视化
- 顶层显示整体进度环形图
- 模块卡片显示线性进度条
- 专题列表使用颜色标记状态（灰色/蓝色/绿色）

### 快速定位
- 面包屑导航：课程 > 模块 > 专题 > 知识点
- 左侧目录树（桌面端），支持点击跳转

### 响应式适配
- **移动端**：单列卡片布局，全屏展开详情
- **平板端**：两列卡片布局
- **桌面端**：三列卡片布局 + 左侧固定目录树

## 1.4 知识状态管理

### 学习状态分类
- 未开始（灰色图标）
- 进行中（蓝色图标，显示进度百分比）
- 已掌握（绿色图标）
- 需复习（橙色图标，根据遗忘曲线算法）

### 状态管理机制
- 支持手动标记学习状态
- 系统自动更新状态（完成练习后自动标记）

## 1.7 技术实现要点

- 使用树形数据结构组织知识体系
- 采用虚拟滚动优化大量知识点渲染性能
- 状态管理使用 Vuex/Pinia（Vue）或 Redux（React）
- 响应式设计使用 CSS Grid/Flexbox 布局
