# 6. 知识扩展

## 功能定位
在学习过程中提供相关知识点链接,帮助学生建立知识网络,拓展思维广度。

## 6.1 知识关联类型

### 前置知识
- 学习当前知识前需要掌握的基础
- 点击可快速复习前置知识
- 检测前置知识掌握程度,不足则提示先学习

### 后续知识
- 基于当前知识延伸的高级内容
- 激发学习兴趣,展望未来学习内容
- 提供预告性介绍

### 相关知识
- 同一主题的不同解题方法
- 同一知识点在不同场景的应用
- 跨章节的知识联系

## 6.2 知识图谱导航

### 可视化知识网络
- 节点表示知识点
- 边表示关联关系
- 颜色表示掌握程度
- 大小表示重要性

### 智能推荐
- 根据当前学习进度推荐下一步
- 标记学习瓶颈知识点
- 推荐最优学习路径

## 6.3 扩展资源

### 视频讲解
- 知识点视频教程
- 经典题目讲解视频
- 方法论专题视频

### 拓展阅读
- 知识点历史背景
- 数学家的故事
- 知识点的实际应用

### 趣味挑战
- 奥数题库
- 数学游戏
- 开放性问题
