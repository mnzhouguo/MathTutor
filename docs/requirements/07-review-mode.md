# 7. 复习模式

## 功能定位
基于艾宾浩斯遗忘曲线,智能规划复习时机,确保长期记忆保持。

## 7.1 复习算法

### 遗忘曲线模型
```
记忆保留率 = e^(-t/S)
```
- t: 距离最后一次学习的时间
- S: 记忆强度(根据学习效果动态调整)

### 复习时机规划

**初次学习后**
- 30分钟后: 第一次复习
- 12小时后: 第二次复习
- 1天后: 第三次复习
- 2天后: 第四次复习
- 4天后: 第五次复习
- 7天后: 第六次复习
- 15天后: 第七次复习

**动态调整**
- 答对 → 延长下次复习间隔
- 答错 → 缩短下次复习间隔,重新开始复习序列
- 连续答对3次 → 标记为"已掌握",减少复习频率

## 7.2 复习模式类型

### 快速复习
- 展示知识点核心要点
- 快速过一遍题目
- 适合时间碎片化场景

### 深度复习
- 重新学习方法论
- 做变式题巩固
- 适合周末或假期

### 错题复习
- 只复习之前做错的题目
- 分析错误原因
- 确保不再犯同样错误

## 7.3 复习提醒

### 提醒方式
- 应用内通知
- 邮件提醒
- 微信/短信推送(可选)

### 智能提醒
- 根据用户学习习惯选择提醒时间
- 避免在深夜或忙碌时段提醒
- 支持自定义提醒计划

## 7.4 复习效果跟踪

### 复习成功率
- 每次复习的答题正确率
- 复习后记忆保留时间
- 复习次数与掌握度的关系

### 复习建议
- 哪些知识点需要优先复习
- 最佳复习时长建议
- 复习频率优化建议
