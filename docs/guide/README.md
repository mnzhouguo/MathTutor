# MathTutor 项目文档

## 项目概述

MathTutor 是一个基于 AI 技术的初中数学 1 对 1 智能辅导系统，旨在帮助学生学习数学概念和解决问题。

## 功能特性

- AI 智能辅导系统
- 个性化学习路径
- 实时互动答疑
- 知识图谱可视化
- 智能复习提醒
