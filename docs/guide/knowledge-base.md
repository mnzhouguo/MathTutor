# 知识库结构说明

## 数据模型

知识库采用分层结构组织，包含以下层级：

### 1. 课程（curriculum）

如"初中数学七年级上册压轴题体系"

### 2. 模块（module）

按知识领域划分的大模块

- `module_id`: 模块唯一标识
- `module_name`: 模块名称
- `module_tag`: 模块标签（如"代数思维篇"）
- `overview`: 模块概述
- `topics`: 包含的主题列表

### 3. 主题（topic）

具体知识点专题

- `topic_id`: 主题唯一标识
- `topic_name`: 主题名称
- `alias`: 别名（如"专题1"）
- `knowledge_points`: 包含的知识点列表

### 4. 知识点（knowledge_point）

最小的知识单元

- `kp_id`: 知识点唯一标识
- `kp_name`: 知识点名称
- `detail`: 详细说明
