# 技术架构设计

## 概述

本文档集合包含 MathTutor 系统的技术架构设计文档，涵盖各模块的详细设计方案。

---

**技术栈**：
- 后端：Python FastAPI
- 前端：React 18 + TypeScript
- 数据库：SQLLite
- OCR服务：百度OCR API
- AI服务：Deepseek / 千问max
